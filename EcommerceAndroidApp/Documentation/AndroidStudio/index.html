About the Files

E-Commerce Android App Template is a mobile e-commerce or online shop template system that used for managing your online shop and many product which you can sell using this app. Admin can manage the order, your product category and menu and other configurations. such as tax and currency code from admin panel on the web and users will be able to see the menu from mobile app and make a order. Run under Android platform which is the number one popular mobile operating system right now.
 
After you buy this item you will get project file of Android app template code, apk file, admin panel, sql database and also this documentation file to guide you installing this app.

Getting Started

Before doing installation, you need to do the following things :
  1. Have a web hosting to store admin panel files and SQL database. Because this documentation using cpanel, a web hosting with cpanel is recommended, if you want work with offline mode, you can use local webserver like xampp, ect with latest version
  2. Install tools that requires for developing Android app such as Java Development Kit and Android Studio as the Official Android IDE. For updated tutorial about how to install them, you can see and Download from Android Developer website.
  3. If you do all things above, you can go to the next step.

How to Import the Project in Android Studio?

  1. Open Android Studio > Import Project (Eclipse ADT, Gradle, ect.)
  2. New window will opened > Browse to directory where you place "EcommerceAndroidApp" project  > Ok
  3. Wait for few minutes until all process import has finished. 

How to Rename Application Package Name?

  1. First click once on your package and then click setting icon on Android Studio.
  2. Close / Unselect Compact Empty Middle Packages
  3. Then, right click your package and rename it.
  4. Right click on the root folder of your project.
  5. Click "Open Module Setting".
  6. Go to the Flavours tab.
  7. Change the application ID with same name of your package name before > Press OK.  

 

How to Change Application Name?

  1. Open Android Studio > app > res > values > strings.xml
  2. Select 'strings.xml'
  3. Enter your app name inside "app_name" string tag :
  4. <string name="app_name">E-Commerce Android App</string>

How to Change the type of Shipping?

  1. Open Android Studio > app > res > values > array.xml
  2. Select 'array.xml' tab next to 'Resources' tab
  3. Enter or add your item array code inside "shipping_array" array tag :
  4. <string-array name="shipping_array">
            <item>POS</item>
            <item>JNE</item>
            <item>TIKI</item>
            <item>COD</item>
        </string-array>
  5. Open Android Studio > app > java > yourpackagename > ActivityCheckout.java Class
  6. Open ActivityCheckout.java
  7. Now, change your shipping type with the same value in array.xml in this following code :
  8. case 0 :
            edtName2.setText("POS"); //POS
            break;
    case 1 :
            edtName2.setText("JNE"); //JNE
            break;
    case 2 :
            edtName2.setText("TIKI"); //TIKI
            break;
    default :
            edtName2.setText("COD"); //COD
            break;
 

 

How to Configure Database and Webservice In Your Server?

Database creation

  1. Open phpmyadmin
  2. Create database and give database name like you want, as example : 'ecommerce' 
  3. Import database ecommerce.sql from donwload package inside 'db' folder

Database Configuration

  1. After creating database configure the connection file, open 'variables.php' in folder 'includes' which is in the includes folder of your package.
  2. That’s all, now run the webservice : http://localhost/ecommerce or with your domain name address
  3. configure your database setting :
  4. $host ="localhost";
    $user ="your_mysql_username";
    $pass ="your_password";
    $database = "your_database_name";
     
  5. If you upload your php code to root folder, as example www.yourdomain.com, please open 'connect_database.php' and 'close_database.php'  in folder 'includes', from this line code below : 
    include($_SERVER['DOCUMENT_ROOT'].'/ecommerce/includes/variables.php'); 
     
    Change with this :
    include($_SERVER['DOCUMENT_ROOT'].'/includes/variables.php'); 
     
    But if you upload your php code into your directory, as example : www.yourdomain.com/ecommerce/,
    just keep the code and don't change it.
    include($_SERVER['DOCUMENT_ROOT'].'/ecommerce/includes/variables.php'); 
     
    'ecommerce' is indicated the name of your directory, if you change to another name, you should change the name in the code become your own directory name.

How to Change the Application Icon?

  1. Open Android Studio > app > res > drawable > ic_launcher.png
  2. Rename your icon with ic_launcher.png and paste > replace in drawable-hdpi folder 
  3. Do the same with on /drawable-mdpi/, /drawable-xhdpi/, and /drawable-xxhdpi/ 

How to change Parse Push notification Application Id?

  1. First open parse.com & Register and Login with your Account.
  2. Press button Create New App in Right Side.
  3. Enter Your App Name
  4. Copy Application Id, Client key.
  5. Open > res > values > strings.xml
  6. Enter your Parse Application Id inside "parse_application_id" string tag
  7. Enter your Parse Client Key Link inside "parse_client_key" string tag
